# integration-project-fresh
Projeto Integrador para o Bootcamp Java do Meli wave 7 - Grupo 1

## Sobre o requisito 6
  Este requisito teve como objetivo criar uma nova funcionalidade que possibilite o comprador a fazer uma review com nota e comentário sobre o produto comprado por ele. A review na página do produto serve como um feedback para o vendedor bem como auxilia outros compradores que estejam interessados nesse produto.

## Prerequisites
1. Spring-boot version: `2.7.4`
2. Dependencies Spring-boot:
- `spring-boot-starter-web`,
- `spring-boot-devtools`,
- `lombok`,
- `spring-boot-starter-test`.
- `spring-boot-starter-validation`.
3. Java version: `Java 11.0.4`.
